# Run Locally

Clone the project

```bash
  git clone https://github.com/SyedaMahamFahim/Netflix-Clone-MERN.git
```

Go to the project directory run backend by 

```bash
  cd frontend
```

Install dependencies

```bash
  npm install
```


Start the server

```bash
  npm start
```
